Inline_response_200_1

  • processed : Boolean
    A flag indicating whether the submitted meter reading has been successfully processed and is valid for update operations. A value of `true` signifies a successful processing, whereas `false` indicates a failure or rejection of the data.
  • reading : BigDecimal
    Confirms the meter reading value in watt-hours (Wh) that was submitted, provided the `processed` flag is `true`.
