This is the API

apiReadingPost

Clients such as energy management systems or smart meter gateways can use this endpoint to push new meter readings to the STROMDAO EAF system at a high frequency. This data is critical for accurately tracking energy consumption and ensuring that customers are billed correctly based on their energy usage patterns. To submit a reading, the client should send a POST request to the /api/reading endpoint with the JSON object containing the meterId, time, and reading.

URL

http://localhost:3001//api/reading

HTTP Method

POST

Response Type

inline_response_200_1

Parameters

  • Body: body Metering.updateReading(Metering.updateReading)
